# ethio-receipt-verify

> **This library has been merged into [cheki](https://github.com/1RB/cheki).**
> 
> cheki is the free, open-source Ethiopian receipt verification platform.
> The Python library now lives at [`/python`](https://github.com/1RB/cheki/tree/main/python) inside the cheki monorepo.
> 
> ## Install
> 
> ```bash
> pip install git+https://github.com/1RB/cheki.git#subdirectory=python
> ```
> 
> ## Why the merge?
> 
> - One repo, one install, one source of truth
> - Web UI + API + CLI + Python library + Docker in one place
> - No more maintaining two repos for the same code
> 
> ## Old install (still works but deprecated)
> 
> ```bash
> pip install git+https://github.com/1RB/ethio-receipt-verify.git
> ```
>
